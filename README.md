# sorairo

## サイト概要
### サイトテーマ
<空の写真をとって投稿する>

### テーマを選んだ理由
<snsとは、不特定多数の人達に情報を発信して、
不特定多数の人達と繋がることを目的としているので、
空を通じて世界中の人達と繋がれるものにしたいと思いました。>

### ターゲットユーザ
<空が好きな人達や、旅行などが好きな人達に使っていただきたいです。>

### 主な利用シーン
<日常や旅先で空の写真を撮って投稿したり、
時間帯で変わる空の色を撮って投稿したり、
変わった形の雲の写真を何か別のものに例えてみたりしてほしいです>
